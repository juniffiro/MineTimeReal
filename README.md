## Getting started

1. Download the latest build from releases
2. Copy to server plugins folder
3. Setup

## Default config

```yaml
settings:
  enabledPlugin: true
  worlds:
    world:
      worldName: world
      timeZone: Europe/Kiev
```

## Commands

| Command   | Description   |
|-----------|---------------|
| /rtreload | Reload config |
